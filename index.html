About

This is a kodi addon repo that allows you to download addons that play videos from the sites below in your android/android tv
Langlangbay (Chinese/Korea/Japanese drama with chinese sub)
GoGoAnime (Animes with English sub)
KShowOnline (Korean Variety Shows with English sub)

Installation

1) Install kodi from play store

Automatic update method (For Kodi 19)

2) Download repository.cywteow.zip transfer into your android/android tv OR add https://cywteow.github.io/repo as source in your kodi file manager
3) Open Kodi -> Settings -> System -> Add-ons -> Enable "Unknown Sources"
4) Kodi Main Page -> Add-on -> Addon browser -> Install from zip file -> select the zip file you downloaded
5) Kodi Main Page -> Add-on -> Addon browser -> Install from repository -> cywteow Add-ons -> Video Add-ons -> Choose any addon to install

Manual update method (When there is an update or the add-ons is not working, you have to come back here to find new .zip file and repeat these steps)

2) Download LangLangBay or GoGoAnime or KShowOnline and transfer into your android/android tv
3) Open Kodi -> Settings -> System -> Add-ons -> Enable "Unknown Sources"
4) Kodi Main Page -> Add-on -> Addon browser -> Install from zip file -> select the zip file you downloaded

If you are still on Kodi 18 and do not wish to update, you can visit my repo and look for those 1.x.x.zip and follow the manual update method

How to see chinese words in kodi (For Langlangbay User)

Go to Kodi system setting -> Interface -> Skin -> Fonts -> Arial based
